# Análisis de Precios de Combustibles en Colombia (2015-2018)

## Descripción
Análisis exploratorio y limpieza de datos de precios de combustibles (Gasolina, Diesel, Kerosene) reportados por estaciones de servicio en Colombia.

### Objetivo
- Limpiar y estructurar la base de datos
- Realizar análisis exploratorio
- Visualizar tendencias de precios por marca, departamento, municipio y tipo de combustible

## Tecnologías utilizadas
- Python
- Pandas
- Plotly (visualizaciones interactivas)
- Jupyter Notebook

## Archivos del proyecto
-  → Notebook principal con todo el análisis
-  → Datos originales
-  → Datos limpios

## Gráficos principales
- Precios por marca y tipo de combustible
- Evolución temporal de precios
- Departamentos y municipios más caros
- Distribución de precios
